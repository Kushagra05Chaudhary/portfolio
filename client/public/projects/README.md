# Project Image Gallery

Add project screenshots in these folders:

- `/public/projects/askren/`
- `/public/projects/voip/`
- `/public/projects/ai-plant/`

Current gallery file names used by the app:

- `askren/askren-1.svg`, `askren/askren-2.svg`
- `voip/voip-1.svg`, `voip/voip-2.svg`
- `ai-plant/ai-plant-1.svg`, `ai-plant/ai-plant-2.svg`

You can replace these `.svg` files with `.png`/`.jpg` and then update paths in `src/pages/Projects.jsx`.
